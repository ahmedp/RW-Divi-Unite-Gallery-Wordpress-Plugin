RW Divi Unite Gallery provides Unite Gallery Custom Module for Divi theme and visual page builder by elegantthemes. 

== Description ==
RW Divi Unite Gallery is all in one  most advanced galley plugin. RW Divi Unite Gallery  gives you facility to create different gallery types which are available in unitegallery like tiles, tiles grid, carousel, compact, grid and slider.  It creates mobile friendly responsive gallery with various skins. RW Divi Unite Gallery plugin is work on most popular divi theme and visual page builder by elegantthemes.  It is easy to use and extendable as well as comes with high performance. You can create gallery just by drag and drop without touching the code.  

RW Divi Unite Gallery gives elegant options to tweak gallery. 

RW Divi Unite Gallery is developer friendly and multilingual. There are many hooks available in code, So you can easily extend and manipulate gallery functionality. 

Credits:
RW Divi Unite Gallery uses unite gallery javscript version which is on http://unitegallery.net/

== Installation ==
= Automatic installation =

To do an automatic install of RW Divi Unite Gallery, log in to your WordPress dashboard with admin credentials, navigate to the Plugins menu and click Add New.

In the search text type “RW Divi Unite Gallery,” and click Search Plugins. Once you’ve found this plugin please click “Install Now” button which will install plugin. Then click on \"Activate\"to activate this plugin.

= Manual installation =

The manual installation method involves downloading this plugin and uploading it to your web server via your favourite FTP application like filezilla. For more information please visit (https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
